Por que não consigo configurar Rede Mesh no Roteador Brisanet?

Guia completo para atendentes de suporte

O Problema

O roteador fornecido pela Brisanet vem pré-configurado e bloqueado para funcionar exclusivamente como roteador conectado via DHCP. Isso significa que as configurações avançadas, incluindo a função de rede mesh via WiFi, não estão acessíveis ao usuário.

O que é Rede Mesh?

É uma tecnologia que permite conectar vários roteadores sem fio para criar uma única rede WiFi ampliada, com cobertura em toda a casa.

Soluções Disponíveis

Opção 1: Usar Cabo Ethernet

Se o cliente quiser manter o roteador Brisanet:

  • Conecte os dispositivos mesh ao roteador Brisanet via cabo ethernet.
  • A conexão cabeada permite que os dispositivos mesh funcionem normally.
  • O roteador Brisanet continuará fornecendo internet via DHCP.
  • Limitação: Requer instalação de cabos pela casa.

Opção 2: Substituir o Roteador

Se o cliente quiser usar mesh via WiFi:

  • O cliente pode substituir o roteador Brisanet por um roteador particular.
  • Conectar o novo roteador à rede Brisanet via DHCP.
  • No roteador particular, o cliente terá acesso completo às configurações.
  • Poderá configurar rede mesh via WiFi sem restrições.
  • Importante: O cliente deve escolher um roteador compatível com mesh.

Resumo Técnico:

O equipamento Brisanet tem firmware proprietário que restringe alterações de configuração para garantir estabilidade da rede. Roteadores particulares oferecem liberdade total de configuração, incluindo recursos avançados como mesh WiFi.
